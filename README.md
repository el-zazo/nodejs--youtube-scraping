# Youtube Scraping

## Video

| Method                      | Description                    |
| --------------------------- | ------------------------------ |
| Get Info                    | like: `title`, `formats`, .... |
| Get Download Link           | get just download link         |
| Get Download Links For Many | get just download links        |

## Playlist

| Method                         | Description                                                 |
| ------------------------------ | ----------------------------------------------------------- |
| Number Videos                  | get number videos in playlist                               |
| Get Info                       | like: `title`, `number_videos`, `videos`, .....             |
| Get Download Links (From - To) | get just download links for a range of videos or all videos |

## Youtube Search

| Method                    | Description                                           |
| ------------------------- | ----------------------------------------------------- |
| Search Videos             | search videos by query                                |
| Search Playlists          | search playlists by query                             |
| Video And Recommendations | get origin video info and here recommandations videos |
